# QuizTempel - Die Quiz-App für Wissenshungrige

**Eine moderne Quiz-App mit Fokus auf Gameplay, Performance und Multiplayer-Erlebnis.**

📲 **iOS Download:**
https://apps.apple.com/de/app/quiztempel/id6745153485

---

## Features

* **Große Fragenvielfalt** aus verschiedenen Kategorien
* **Singleplayer & Multiplayer** (1v1 Gameplay)
* **Echtzeit-Spielsystem** mit abwechselnden Zügen
* **Ranglisten & Statistiken** zur Fortschrittsverfolgung

---

## Technologien

* **iOS (Swift, SpriteKit)**
* **Supabase (PostgreSQL, Auth, Realtime)**
* **Realtime Multiplayer Architektur**

---

## Projektüberblick

QuizTempel ist eine eigenständig entwickelte iOS-App mit Fokus auf:

* saubere Architektur
* performantes State-Management
* skalierbares Backend für Multiplayer-Games

Das Projekt demonstriert insbesondere:

* Realtime-Datenverarbeitung
* Multiplayer-Logik (Turn-Based System)
* Backend-Integration mit Supabase
